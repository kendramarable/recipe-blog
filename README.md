# Next.js Recipe Blog

- blogposts are statically generated (all built at build time, don't have to re-fetch when loading pages)
- all recipes are server-side rendered, improving SEO

Website design is still being worked on

## How to Run Locally

1. navigate to this app's directory
2. run the following command: npm run dev
3. go to your web browser and access via the link given (will likely be localhost:3000)
