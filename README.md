# Cindy's Cookbook

## This application is hosted at https://cindys-cookbook.netlify.app/

## How to Run Locally

1. navigate to this app's directory
2. run the following command: npm run build
3. start the application by running the following command: npm run start
4. go to your web browser and access via the link given (will likely be localhost:3000)

Note: if running into any errors while building, try deleting the generated .next folder and re-building
